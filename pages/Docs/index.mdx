# Setting up the oslash mono-repo

#### Step 1 - Clone the repo
Clone the oslash mono-repo to your local machine. Run the following command in a terminal

```git clone https://github.com/realtimeskills/oslash.git```

#### Step 2 - Change directory
Change your working directory to the mono-repo directory

```cd oslash-main-repo```

#### Step 3 - Installing the dependencies
In order to install all the dependencies required to run the project we need to run the following command in the terminal

```npm install```
