# OSlash Launcher


### Step 1:
Clone the [OSlash Monorepo](https://github.com/realtimeskills/oslash).

### Step 2:
Use the given command to install all the dependencies for the project to run.
```
npm i 
or
npm install
```

### Step 3:
Redirect to the Launcher folder using 
```
cd frontend/launcher/
```

### step 4:
Run the program using 
```
npm start
```
After running the above command, it will redirect us to the [Launcher Page](https://home.oslash.com/launcher).
